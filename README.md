# Toast工具

* 用于处理某些国产手机Toast无法显示的问题，自定义Toast，显示提示信息
* 汇集Toast提示工具类，方便调用

<br>

[![Download][icon_download]][download]

```
compile 'com.excellence:toast:_latestVersion'
```
